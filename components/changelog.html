1.0.0 - 6/16/23

  • Initial Release

1.0.2 - 6/17/23

  • Improved dox - fixed bugs
  • Fixed components would not render with emmet expression: {{}}
  • Added 
    <import src="file" exports="name of export u want to render!"></import>
     tag to import components within components!
  • Improved html var tag and made it usable any code within can be used as a variable and returned as {{var_name}} 
    <var name="somename">
     tag

1.0.3 [CURRENT] - 6/18/23

  • Improved dox - fixed more bugs
  • Fixed route issues where imported elements would not render when sequentially rendered
  • Router now allows you to set routes in the meta tag 
    <meta config routes="/home,/changelog,/car">
    also be sure to set fallback route in the meta tag 
    <meta config fallback="/home">
  • Added $import & $export to dox - now you can import and export jsdox components 
    $import('file' || 'exportName')
     
    • $export('exportName', objectvalue)
